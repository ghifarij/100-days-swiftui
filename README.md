# 100 Days of SwiftUI

This repository documents my journey through **100 Days of SwiftUI** by building small projects and learning concepts through hands-on practice. More projects will be added over time.

---

## Projects

### 1. WeSplit

**What it does:**
A bill-splitting app that calculates how much each person should pay, including tip.

**What I learned:**
State management, forms, input handling, basic calculations.

---

### 2. Unit Converter

**What it does:**
Convert between length, time, and temperature units.

**What I learned:**
View composition, conditional UI, unit conversion logic, state handling.
